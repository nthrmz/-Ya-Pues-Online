# ¡Ya Pues! App Companion

## 1. Título del Proyecto
**Ya Pues! App Companion**

## 2. Descripción General
Aplicación web companion para el juego de mesa físico **¡Ya Pues!**, que facilita el control de tiempo, el orden de argumentación y la experiencia visual durante las partidas.

## 3. Problema que Resuelve / Qué Simula
La web apoya a los jugadores del juego físico al digitalizar funciones auxiliares del juego:
- control de tiempos de debate,
- sorteo del orden de argumentos,
- registro de ganadores por ronda,
- y presentación de reglas con soporte visual.

## 4. Tecnologías Utilizadas
- HTML
- CSS
- JavaScript
- Bootstrap 5
- LocalStorage
- Animaciones CSS
- Diseño responsive

## 5. Uso de Herramientas de IA
El proyecto se ha desarrollado con apoyo de herramientas de productividad como **GitHub Copilot** para sugerencias de código y organización.

## 6. Funcionalidades
- Temporizador de debate configurable.
- Registro de nombres de jugadores.
- Sorteo aleatorio del orden de argumentos.
- Ranking de ganadores por ronda.
- Sección de reglas paso a paso.
- Sección de merchandising.
- Navegación entre páginas estáticas.
- Modo claro/oscuro.

## 7. Ejemplos de funcionalidades dinámicas
- Sortear orden de jugadores.
- Guardar el ranking de ganadores en el navegador.
- Cambiar entre modo claro y modo oscuro.

## 9. Cómo clonar y ejecutar localmente
```bash
git clone https://github.com/nthrmz/-Ya-Pues-Online.git
cd -Ya-Pues-Online
# Abrir en el editor o usar Live Server para ejecutar
```
Luego abrir `index.html` en el navegador o usar una extensión como Live Server en Visual Studio Code.

## 10. Estructura del Proyecto
- `index.html`: página principal.
- `como-se-juega.html`: explicación de reglas y juego físico.
- `simulador.html`: temporizador y ranking.
- `merchandising.html`: sección de productos gráficos.
- `foro.html`: sección de comentarios.
- `assets/css/`: estilos del proyecto.
- `assets/js/`: scripts de interacción.
- `recursos/`: imágenes y recursos gráficos del juego.
