¡Ya Pues! App Companion — versión estilizada

Enfoque:
App de apoyo para el juego físico de mesa. No promete multijugador online real.
Incluye inicio, reglas, temporizador/sorteo, merchandising y foro.

Cambios destacados:
- Header con logo a la izquierda, navegación al centro y botón para empezar a jugar.
- Modo claro/oscuro con localStorage.
- Footer más llamativo con logo blanco, marquee y formas animadas.
- Apartado independiente de Merchandising.
- Inicio explica que el juego fomenta el turismo en Cochabamba.
- Temporizador enfocado en apoyo al juego físico: nombres, personajes, tiempo, sorteo de argumentación y ranking.
- Foro con comentarios, tipos, buscador, filtros y reacciones.
- Animaciones al hacer scroll y transición entre páginas.

Cómo probar:
1. Abrir index.html con Live Server / Go Live.
2. Revisar Inicio, Cómo se juega, Temporizador, Merchandising y Foro.
3. Para subir a GitHub, copiar todo dentro de la carpeta enlazada y luego ejecutar:
   git add .
   git commit -m "Ajustes finales app companion Ya Pues"
   git push
